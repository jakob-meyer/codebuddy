## Features

### SVG to CSS

Minimizes a SVG using [SVGO](https://github.com/svg/svgo) and converts it to a url encoded CSS string.

## Development

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
